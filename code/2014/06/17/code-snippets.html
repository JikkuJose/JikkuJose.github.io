Code Snippets

I imagine this blog to have a significant number of code snippets and this very post is an attempt to find a solution for it.

And I found it! Not only was the solution easy, switching to a custom theme was also a cake walk. Take a look at the code used to generate the highlighted snippet:

{% highlight ruby %}
def show
  @widget = Widget(params[:id])
  respond_to do |format|
    format.html # show.html.erb
    format.json { render json: @widget }
  end
end
{% endhighlight %}

..will produce:

def show
  @widget = Widget(params[:id])
  respond_to do |format|
    format.html # show.html.erb
    format.json { render json: @widget }
  end
end
